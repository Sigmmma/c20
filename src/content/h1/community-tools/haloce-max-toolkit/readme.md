The **Halo CE Max Toolkit** is a suite of tools for modern [3ds Max](~3dsmax) versions which support importing and exporting source and tag files for Halo 1. The project consolidates and rewrites legacy importers and exporters, such as [Bluestreak](~obsolete#bluestreak), offering improvements and new features.

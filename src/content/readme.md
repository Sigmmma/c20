Welcome to **The Reclaimers Library** (**c20**), a project which aims to document and centralize the immense tribal knowledge of the Halo modding community. This is an unofficial volunteer resource by modders, for modders.

We focus on covering main line Halo games with official modding tools releases: Halo Custom Edition and the new MCC [mod tools](~general/mod-tools) for H1, H2, H3, ODST, HR, H4, and H2A. We do not explicitly cover modding on Xbox, Halo Online, [ElDewrito](https://www.eldewrito.org/), Halo 2 Vista/[project cartographer](https://www.cartographer.online/), or [Halo Wars](https://halowarsmodding.github.io/) though information on this site may still be useful.

_Mods_ are fan-made modifications to a game that change or add content, or implement unofficial bug fixes and new features. Modding is a great way to learn new skills and put your own spin on the game! Halo has a tradition of modding since the early 2000s which is still going strong. On c20 we focus on today's best practices and tools.
